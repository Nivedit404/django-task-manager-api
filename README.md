# Task Manager API

A RESTful API built with Django for managing tasks with authentication and CRUD operations.

## Tech Stack
- Python
- Django
- SQLite
- Docker
- Postman

## Features
- User authentication
- Create, read, update, delete tasks
- RESTful API endpoints

## Setup Instructions
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run server:
   python manage.py runserver
